Earthquake Maps

About Earthquake

The USGS provides earthquake data in a number of different formats, updated every 5 minutes. Using the data available on www.earthquake.usgs.gov created maps using Leaflet that plots all of the earthquakes from the data set based on their longitude and latitude.

The data markers reflect the magnitude of the earthquake in their size and color. Earthquakes with higher magnitudes should appear larger and darker in color.

Popups provide additional information about the earthquake when a marker is clicked and the legend provides context for the data.

Layer map has different layer options such as outdoors, satellite, grayscale and light map

Cluster map shows the number of earthquakes in each cluster and as we zoom it adjusts the clusters based on the currret view.

Heatmap highlights areas of earthquake activity.
